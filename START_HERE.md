## 🎓 How It Works (Simple Explanation)

```
1. User opens login.html
   ↓ Creates account with email & password
   
2. Backend hashes password with bcryptjs
   ↓ Saves to MongoDB
   
3. User logs in with credentials
   ↓ Backend verifies password
   ↓ Generates JWT token
   
4. Token stored in browser's localStorage
   ↓ Used for all future API calls
   
5. User opens dashboard
   ↓ Sends token with every request
   
6. Backend verifies token
   ↓ Extracts user ID from token
   ↓ Returns ONLY that user's data
   
7. Frontend displays bills & cash
   ↓ User adds/edits/deletes data
   ↓ Saved to MongoDB automatically
   
8. Stats calculate in real-time
   ↓ Based on database data
   
9. Logout clears token
   ↓ User must login again
```
